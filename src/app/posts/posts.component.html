About Github search

Github Search is basically an app that helps users to search for other users by their github usernames.

How to search a github user using this app:

Navigate to the users page throught he sidabar by clicking the users icon

Here there is a blank search field for seaching for github users.

Input the user's github username and it brings the user's information.

You can also view the user's github page by clicking the button that comes in the user's information./p>
